**%status:** Shows if the server is online, and players in the server

**%craft <item>**: Shows the crafting recipe for the item

**%wiki <input>:** Shows the Minecraft Wiki page for the given input

**%paid:** Shows who has NOT paid this month

**%update:** Shows the latest update to the bot
